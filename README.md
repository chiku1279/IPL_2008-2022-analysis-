# 🏏 IPL Analytics Dashboard (2008–2022) | Power BI Project

## 📌 Overview
This project presents an IPL Analytics Dashboard developed in Power BI to analyze historical Indian Premier League data from 2008 to 2022. The dashboard converts raw match and ball-by-ball datasets into interactive visual insights, helping users evaluate team performance, player statistics, venue trends, and season-wise outcomes.

The dashboard is designed to provide a complete analytical view of IPL history by combining match statistics, player achievements, and strategic insights into one data-driven solution.

---

## 🎯 Objective
The primary objective of this project is to:
- Analyze IPL performance across multiple seasons  
- Compare team consistency and championship success  
- Track player achievements such as Orange Cap and Purple Cap winners  
- Study toss impact and match-winning strategies  
- Evaluate venue and city performance trends  
- Build an interactive dashboard for cricket analytics and storytelling  

---

## 📊 Dashboard

## 🧹 Data Cleaning & Transformation
To ensure data accuracy and dashboard efficiency, the following transformations were performed:
- Removed null values and duplicate records  
- Standardized inconsistent team names  
- Corrected date and season formats  
- Merged match and delivery tables  
- Created calculated columns for:
  - Match Result  
  - Win Margin  
  - Super Over Indicator  
- Built calendar hierarchy for time-based analysis  

---

## 🔗 Data Modeling
A structured data model was created using relationships between:
- Matches Table  
- Deliveries Table  
- Calendar Table  
- Team Dimension  
- Player Dimension  

### Modeling Strategy:
A star schema approach was used to improve dashboard performance and analytical flexibility.
